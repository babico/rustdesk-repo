Quick Setup

1

Import the signing key

curl -fsSL https://babico.github.io/rustdesk-repo/apt-repo.gpg | sudo gpg --dearmor -o /usr/share/keyrings/personal-apt.gpg
2

Add the repository

echo "deb [arch=amd64 signed-by=/usr/share/keyrings/personal-apt.gpg] https://babico.github.io/rustdesk-repo stable main" | sudo tee /etc/apt/sources.list.d/personal-apt.list
3

Install any app

sudo apt update
sudo apt install rustdesk              # RustDesk
sudo apt install mattermost-desktop    # Mattermost Desktop
